## Summary

This release adds single device-login controls for better account security, updates the backend framework for improved performance, and includes refreshed login and signup page designs. You'll also find improved upload handling and better mobile search experience.

## What's New

- Adds option to allow only one logged-in device per user account at a time
- Adds landing page toggle option
- Adds Google Analytics JSON credentials support

## Enhancements

- Updates backend framework requirement for improved server performance
- Refreshes login and signup page designs
- Improves admin debugging for server errors
- Improves upload handling and cloud storage support
- Unifies email templates across the platform
- Improves general settings validation
- Improves embed link handling
- Improves mobile search user experience

## Bug Fixes

- Fixed local and YouTube search issues
- Fixed genre name handling problems
- Fixed playlist ownership display errors
- Fixed track SEO fallback issues
- Fixed cache method validation errors
- Fixed cache clearing problems
- Fixed popularity sorting issues
- Fixed text logo overflow issues
- Fixed user background removal problems
- Fixed custom CSS and JS storage issues
- Fixed SEO tag placeholder handling
