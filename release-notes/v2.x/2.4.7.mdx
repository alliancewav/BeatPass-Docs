## Summary

This release improves the upgrade process when updating from older versions and enhances email configuration reliability. We've also transitioned to a more flexible user role system for managing permissions.

## What's New

- Improves upgrade process when upgrading from versions older than 2.4.6

## Enhancements

- Replaced subscription-based access controls with user role system for more flexible permission management

## Bug Fixes

- Fixed email configuration validation issues
- Fixed older channel editability problems
- Fixed outdated email method issues
- Fixed Google Analytics validation errors
- Fixed legacy image loading problems
