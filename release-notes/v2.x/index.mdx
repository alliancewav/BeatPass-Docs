## Key Features in v2.x

<AccordionGroup>
  <Accordion title="Collaborative Playlists (v2.4.9)" icon="users">
    Multiple users can contribute to shared playlists with moderation controls.
  </Accordion>
  <Accordion title="Full API (v2.4.9)" icon="code">
    Complete REST API with token management for third-party integrations.
  </Accordion>
  <Accordion title="Spotify Integration (v2.4.9+)" icon="music">
    Import artists, albums, and tracks directly from Spotify.
  </Accordion>
  <Accordion title="Cloud Storage (v2.4.5+)" icon="cloud">
    Direct browser uploads to cloud storage providers.
  </Accordion>
  <Accordion title="GDPR Compliance (v2.4.5)" icon="shield">
    Cookie notices, consent flows, and data handling compliance.
  </Accordion>
</AccordionGroup>
