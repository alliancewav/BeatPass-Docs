## Summary

This release adds automatic lyrics extraction from uploaded track metadata, new permissions for embedding and audio playback, and various user experience improvements for mobile users and playlist management.

## What's New

- Adds automatic lyrics extraction from uploaded track metadata
- Adds new permissions for embedding tracks and playing audio
- Adds mobile account menu item in controls by default
- Adds queue entire track lists to player, similar to track tables

## Enhancements

- Improves mobile edit layouts
- Improves playlist ordering behavior
- Improves fullscreen local video handling
- Improves dark mode scrollbars
- Updates data tables for better performance

## Bug Fixes

- Fixed queue updates in fullscreen mode
- Fixed track list sorting issues
- Fixed liked track filters
- Fixed media file deletion
- Fixed homepage channel loading
- Fixed upload reliability
- Fixed scrolling behavior
- Fixed notification URL handling
- Fixed various navigation and layout issues
