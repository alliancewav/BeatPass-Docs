## Summary

This release brings admin quality-of-life improvements and reliability fixes for comments, channels, and permissions. You'll find better comment management, automatic channel cache clearing, and various fixes for queue behavior and permissions.

## Enhancements

- Improves admin comments table to show the album or track each comment belongs to
- Adds automatic channel cache clearing when a channel is edited or auto-updated
- Adds ability to manually enter Spotify IDs for artists, albums, and tracks
- Updates admin data table design with additional filters

## Bug Fixes

- Fixed artist table sorting by album count
- Fixed ability for users to delete their own comments
- Fixed album repost management
- Fixed queue behavior on radio pages
- Fixed album-to-queue behavior
- Fixed backstage artist link removal
- Fixed social login panel visibility to respect permissions
- Fixed SEO tags for albums
- Fixed interface settings tabs behavior
- Fixed embed permissions
- Fixed try pro visibility
- Fixed playlist collaboration rules
