## Summary

This release introduces collaborative playlists, a new backstage section for artist verification, and multiple search options including Meilisearch and Elasticsearch. You can now also import content from Spotify, use embeddable playlists, and access a full API for integrating BeatPass with other applications.

## What's New

- Adds collaborative playlists that multiple users can contribute to
- Adds backstage section for claiming, verifying, or becoming an artist
- Adds admin tools for managing artist verification requests
- Adds support for multiple search integrations for faster, more accurate search results
- Adds combined Spotify plus local search provider
- Adds new channel auto-update method for Spotify playlists
- Adds comments management page in the admin area
- Adds ability to import artists, albums, tracks, and playlists from Spotify by ID
- Adds embeddable playlists and Spotify genre radio
- Adds full API for integrating BeatPass with other apps
- Adds API access token management and live API documentation
- Adds verified artist badges and richer track tables with artwork

## Enhancements

- Adds support for multiple artists on albums
- Adds ability to delete comments from track pages
- Increases comment length limits
- Simplifies music permissions
- Adds profile page customization options
- Improves history behavior
- Adds additional lyrics providers
- Adds country and city metadata support
- Improves embed behavior and appearance

## Bug Fixes

- Fixed Spotify channel behavior
- Fixed form handling issues
- Fixed waveform display and seeking behavior
- Fixed embed loading problems
- Fixed installation issues
- Fixed playlist context menu behavior
- Fixed GDPR embed compliance
- Fixed upload behavior reliability
- Fixed artwork display issues
- Fixed navigation warning behavior
