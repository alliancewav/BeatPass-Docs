## Summary

This release improves backend tools for importing content from Spotify and makes channel management more reliable. You'll also find fixes for manual content channels and SEO settings.

## What's New

- Adds ability to import artists and albums by Spotify ID even when using local providers

## Enhancements

- Improves channel creation and management reliability

## Bug Fixes

- Fixed manual content channels when creating new channels
- Fixed manual reordering of channel contents
- Fixed artist and track names in browser tab titles
- Fixed SEO settings saving in the appearance editor
- Fixed profile editing reliability
- Fixed search record importing in settings
