## Summary

This release adds GDPR compliance support, improves large file uploads with pause and resume capability, and introduces cloud storage options for your media files. You can now also use new channel auto-update methods and separate pages for editing your own tracks and albums.

## What's New

- Adds GDPR cookie notice and confirmation policies during registration
- Adds pause and resume support for large file uploads
- Adds channel auto-update methods for popular artists and recently added tracks
- Adds separate pages for editing your own tracks and albums outside the admin area
- Adds tag-based listing pages for tracks and albums
- Adds cloud storage support for music, video, avatars, and other media

## Enhancements

- Improves mini player with double-click video to toggle fullscreen
- Updates automatic channel titles and SEO placeholders when site name changes
- Improves search results when editing channels with manual content update options
- Improves tag and genre suggestions in edit forms

## Bug Fixes

- Fixed validation error visibility issues
- Fixed scrolling behavior problems
- Fixed sitemap generation issues
- Fixed permission handling errors
- Fixed history and chart display reliability
- Fixed various UI and caching issues across the platform
