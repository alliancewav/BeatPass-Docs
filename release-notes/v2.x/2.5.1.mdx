## Summary

This release improves compatibility with newer server software and enhances search provider integration. You'll find better error messaging, more reliable waveform handling, and fixes for social sharing and various other features.

## Enhancements

- Improves error messaging when server software version is incompatible
- Adds ability to toggle artist verified status from the admin area
- Improves track player to always fall back to simple seekbar when waveform is unavailable

## Bug Fixes

- Fixed channel search when content type is set to multiple types
- Fixed redirects after creating artists
- Fixed social sharing images for tracks and albums
- Fixed similar artist links
- Fixed become artist menu behavior
- Fixed backstage artist search
- Fixed mobile library pages
- Fixed sitemap generation
- Fixed Spotify import behavior
- Fixed radio behavior on newer server software
- Fixed search integration reliability
- Fixed image privacy settings for cloud storage
