## Summary

This release brings UI polish and fixes for search, tables, and embeds. You'll find improved artist grid layouts, better mobile search reliability, and corrected track list playback behavior.

## Enhancements

- Centers play button on artist grid items
- Disables (instead of removes) like and repost buttons when viewing your own tracks

## Bug Fixes

- Fixed mobile song search reliability
- Fixed pricing tables display on landing pages
- Fixed player layout issues with long track names
- Fixed overlay close behavior on navigation
- Fixed track list and channel grid playback issues
- Fixed tag manager and indexing issues for analytics and search
- Improved selection and external links behavior in menus and tables
