## Summary

This is a complete platform overhaul with a new frontend built with React for better performance, smaller bundles, and smoother animations. You'll find a refreshed design across most pages, improved admin tools with sitewide statistics, direct browser uploads to cloud storage, and updated billing with Apple Pay and Google Pay support.

## What's New

- Adds sitewide play statistics in the admin area, plus stats per artist, album, and track
- Adds ability to list artists, albums, and tracks of a specific genre in the same channel
- Rewrites frontend with React for better performance and smoother animations
- Refreshes design for most pages across the site
- Adds direct browser uploads to cloud storage providers
- Adds Apple Pay and Google Pay support to billing
- Adds new admin files page with preview support
- Improves email confirmation flows

## Enhancements

- Adds bulk track reordering in playlists
- Improves performance in large track tables
- Improves admin data tables for artists, albums, and tracks
- Improves upload forms with time remaining indicators and better empty states
- Updates landing page design with more layout options and inline pricing tables
- Adds editable admin sidebar menus via the menu manager
- Improves subscription flow
- Adds compatibility with newer server software versions
- Improves appearance editor with more icons and better GDPR controls
- Improves translation coverage

## Bug Fixes

- Fixed player loading and waveform synchronization issues
- Fixed playlist duration calculations
- Fixed chart timezone handling
- Fixed admin table sorting reliability
- Fixed login state handling when changing passwords
- Fixed browser language selection to respect admin configured defaults
- Fixed registration disabling to correctly block API registration routes
