## Summary

This release brings quality-of-life improvements for search and track context actions. You'll now find context menus on search results and fixes for release date extraction, mobile menus, and various other issues.

## What's New

- Adds context menus to track, album, artist, and playlist items in search dropdown

## Bug Fixes

- Fixed release date extraction from uploaded files
- Fixed become author menu items on mobile
- Fixed history page behavior when ad blockers are present
- Improved slug generation for artist names
- Fixed search profiles tab behavior
- Fixed email verification issues
- Fixed lyrics button functionality
- Fixed channel cache invalidation
