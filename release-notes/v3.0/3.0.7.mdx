## Summary

This release brings performance, SEO, and admin user experience improvements plus a large batch of bug fixes. You'll find faster page loading, improved SEO tools, better comment management, and various playback reliability fixes.

## What's New

- Improves first page load speed for listeners
- Improves SEO tags editor in the admin area
- Adds searchable tags for artists and albums
- Adds reported filter for comments management
- Improves visibility of user suspension reasons
- Improves email verification flows

## Bug Fixes

- Fixed HLS playback reliability
- Fixed iOS video behavior
- Fixed album and artist links
- Improved edit flows after claiming artists
- Fixed datepicker behavior and keyboard input
- Fixed appearance editor persistence and logo handling
- Fixed subscription plans and billing issues
- Fixed PayPal subscription handling
- Fixed API access permissions
- Improved mobile search and overlay behavior
- Fixed many smaller layout, dialog, and device-specific issues
