## Summary

This release introduces adaptive streaming support with HLS and DASH formats, improves table responsiveness, and includes stability fixes for permissions, menus, and billing.

## What's New

- Adds HLS and DASH streaming support for track sources
- Adds ability for uploaded tracks to inherit genres, tags, and artists from their album
- Improves table responsiveness across the platform

## Bug Fixes

- Fixed missing permissions and role name translations
- Fixed automation disable behavior
- Fixed menu display on landing pages
- Fixed artist discography pagination
- Fixed pricing savings calculation
- Fixed several smaller layout issues
