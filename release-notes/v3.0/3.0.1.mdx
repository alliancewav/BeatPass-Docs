## Summary

This is a targeted bugfix release addressing issues with search, uploads, channels, and menu management. These fixes improve the reliability of mobile search, album creation, and channel management.

## Bug Fixes

- Fixed space character input in mobile search
- Fixed track playback when the same track appears both individually and as part of an album
- Fixed upload progress display when uploading multiple tracks during album creation
- Fixed default genre channel creation during installation
- Fixed local plus Spotify combined search functionality
- Fixed channel deletion in the menu manager
- Fixed channel URL handling in the menu manager
- Fixed layout issue where long track names pushed options buttons off screen
