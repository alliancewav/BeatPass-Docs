## Summary

This bugfix release focuses on comments, search, and library behavior. You'll find improved comment settings, better search functionality, and smoother library updates.

## Bug Fixes

- Added missing translations to localization tools
- Fixed comments to respect settings and visibility flags
- Fixed search section links
- Improved artist album editing reliability
- Fixed user profile dialogs on mobile
- Fixed library add and remove actions to update without requiring page refresh
- Fixed share playlist functionality
- Fixed channel title translations
- Fixed several smaller interface issues
