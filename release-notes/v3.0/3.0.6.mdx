## Summary

This security-focused release adds two-factor authentication, account suspension controls, and various user experience improvements. You'll find better responsive tables, improved album upload handling, and fixes for mobile and tablet interfaces.

## What's New

- Adds two-factor authentication support for BeatPass accounts
- Adds ability for admins to suspend or ban users
- Adds option to show full-size social login buttons on authentication pages
- Adds ability to block registrations from specific email domains
- Adds active sessions panel in account settings

## Enhancements

- Improves responsive tables across the site
- Improves handling of uploads and validation when creating albums with multiple tracks

## Bug Fixes

- Fixed playlist loading on mobile
- Fixed library menus on tablets
- Fixed checkout page visual issues
- Fixed theme handling and dark mode defaults
- Fixed compatibility with newer search methods
- Fixed admin user handling
